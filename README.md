# 🤖 Vibe Coding Bot

Telegram-бот, который автоматически генерирует и публикует посты об обучении вайбкодингу через Claude AI.

## Что умеет

- 📅 Публикует посты в канал по расписанию (по умолчанию 9:00 и 19:00)
- 🤖 Генерирует уникальный контент через Claude AI
- 📨 Рассылает посты подписчикам в личку
- ✋ Команда `/post` — получить пост вручную прямо сейчас

## Быстрый старт

### 1. Создай бота в Telegram
- Напиши @BotFather → /newbot
- Скопируй токен

### 2. Добавь бота в канал как администратора
- Зайди в настройки канала → Администраторы → добавь бота
- Дай права на публикацию

### 3. Получи ID канала
- Для публичного канала: просто `@имя_канала`
- Для приватного: перешли пост из канала боту @userinfobot

### 4. Получи ключ Claude API
- Зарегистрируйся на console.anthropic.com
- Создай API ключ

### 5. Настрой переменные окружения
```bash
cp .env.example .env
# Заполни .env своими ключами
```

### 6. Установи зависимости и запусти
```bash
pip install -r requirements.txt
python bot.py
```

## Деплой на Railway

1. Загрузи проект на GitHub
2. Зайди на railway.app → New Project → Deploy from GitHub
3. Добавь переменные окружения в Settings → Variables
4. Railway автоматически запустит бота

## Настройка расписания

Измени файл `scheduler_config.py`:
```python
SCHEDULE = [
    {"id": "morning_post", "hour": 9,  "minute": 0},
    {"id": "evening_post", "hour": 19, "minute": 0},
]
```
