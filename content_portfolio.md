## ЛАЙФХАКИ

### Cursor
1. **Tab completion** — выключи автодополнение, включи только Tab. Скорость x3.
2. **Контекст** — прикрепляй файлы через @file, а не копируй код.
3. **Режим Composer** — создавай целые проекты за один промпт.
4. **Undo** — Ctrl+Z откатывает изменения ИИ, не бойся экспериментировать.
5. **Режим Agent** — Cursor сам ищет файлы и исправляет ошибки.

### Claude
1. **Системный промпт** — опиши роль: "Ты — senior React разработчик"
2. **Примеры** — покажи 2-3 примера желаемого результата
3. **Цепочки** — разбивай задачу на шаги, не проси всё сразу
4. **Reframing** — "Объясни как для 5-летнего, потом напиши код"
5. **Контекст** — кидай entire файл, а не фрагмент

### v0.dev
1. **Описывай дизайн** — "Минималистичный лендинг с градиентом"
2. **Копируй компонент** — не весь проект, а отдельные блоки
3. **Комбинируй** — генерируй header, footer, hero по отдельности
4. **Тема** — укажи цвета и шрифты в промпте

### Supabase
1. **Auth** — включи Google/GitHub за 2 минуты
2. **RLS** — всегда настраивай Row Level Security
3. **Edge Functions** — серверная логика без сервера
4. **Realtime** — подписки на изменения в реальном времени

### Vercel
1. **Preview** — каждый PR получает свою ссылку
2. **Env vars** — настраивай в dashboard, не в коде
3. **Edge** — деплой на CDN автоматически
4. **Analytics** — встроенная аналитика, не нужен GA

## КОНТЕНТ-ПОРТФЕЛЬ ( готовые форматы )

### Формат 1: "До/После"
**Структура:**
- Проблема (1-2 строки)
- Решение (что сделал)
- Результат (цифры)
- Мораль (1 строка)

**Пример:**
```
До: Клиент хотел сайт за 100К
После: Сделал за 40 минут, взял 50К
Результат: +50К за вечер
Мораль: Вайбкодинг = деньги
```

### Формат 2: "Инструмент дня"
**Структура:**
- Название
- Что делает
- Как использовать
- Ссылка

**Пример:**
```
🔥 Cursor Rules

Что делает: Настраивает поведение ИИ для твоего проекта

Как использовать:
1. Создай .cursorrules
2. Опиши стек технологий
3. Добавь правила кодирования

Ссылка: cursor.sh
```

### Формат 3: "Промпт-рецепт"
**Структура:**
- Задача
- Промпт (готовый)
- Результат

**Пример:**
```
Задача: Создать лендинг для кафе

Промпт:
"Создай React лендинг для кафе 'Уютное место':
- Header с логотипом и навигацией
- Hero с фоновым фото еды и кнопкой 'Заказать'
- Секция 'Меню' с карточками блюд
- Секция 'О нас' с историей
- Footer с контактами и картой
- Используй Tailwind CSS, цвета: #8B4513, #F5DEB3"

Результат: Готовый лендинг за 10 минут
```

### Формат 4: "Топ-3"
**Структура:**
- Тема
- 3 пункта с объяснением
- Призыв

**Пример:**
```
Топ-3 навыка для вайбкодера:

1. Промпт-инжиниринг — умей объяснять ИИ что хочешь
2. Дизайн-мышление — понимай что нужно пользователю
3. MVP-подход — делай Minimum Viable Product

Начни с первого. Сегодня.
```

### Формат 5: "Миф/Факт"
**Структура:**
- Миф (распространённое заблуждение)
- Факт (правда)
- Доказательство

**Пример:**
```
Миф: "Нужно знать 5 языков программирования"

Факт: Достаточно уметь объяснять что хочешь на русском

Доказательство: Cursor + Claude = готовый проект без единой строки кода
```

### Формат 6: "Чеклист"
**Пример:**
```
Чеклист перед деплоем:

✅ Все тесты проходят
✅ Окружение настроено
✅ .env файл в .gitignore
✅ README обновлён
✅ Код отформатирован
✅ Нет секретов в коде
✅ Домен настроен
✅ SSL-сертификат активен

Отметь галочку когда всё готово!
```

### Формат 7: "История успеха"
**Структура:**
- Человек (кто)
- Проблема (что было)
- Решение (что сделал)
- Результат (цифры)

**Пример:**
```
Алексей, 28 лет, менеджер

Проблема: Хотел зарабатывать больше, но не знал код

Решение: Изучил Cursor за неделю, начал делать лендинги

Результат: 3 клиента × 30К = 90К за первый месяц

Теперь он фрилансит и уволился с работы
```
