## ПОСТЫ НА ОСНОВЕ СКИЛЛОВ

### Пост: Context7
Знал ли ты, что Claude может обращаться к актуальной документации?

Context7 подключается к MCP и даёт Claude доступ к свежим API-референсам.

Больше никаких устаревших ответов.

### Пост: n8n
n8n — это 400+ интеграций без кода.

Подключи Telegram, Google Sheets, GitHub, Stripe и автоматизируй рутину.

Идеально для тех, кто не хочет писать код для автоматизации.

### Пост: Chrome DevTools MCP
Claude теперь может дебаггить твой сайт в Chrome.

Chrome DevTools MCP — AI получает доступ к DevTools.

Находи ошибки в реальном времени.

### Пост: drawio-skill
Опиши диаграмму текстом — Claude нарисует её за тебя.

drawio-skill генерирует диаграммы из естественного языка.

Экспорт в PNG, SVG, PDF.

### Пост: GitHub MCP
Управляй GitHub из Claude Code.

Создавай PR, закрывай issues, пушь код — всё из терминала.

Официальный MCP от GitHub.
